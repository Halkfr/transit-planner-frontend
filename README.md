# Estonian Bus Schedule Web App

A simple **JavaScript + Tailwind CSS** frontend for viewing Estonian bus stop information and schedules.  
Bus stop data is sourced from **peatus.ee GTFS** (`https://peatus.ee/gtfs/gtfs.zip`).

This project was built **without a frontend framework**, as required by the course assignment.

## Live Demo

Frontend is hosted on Azure Static Web Apps:  
https://icy-stone-04c89ca03.2.azurestaticapps.net

---

## Project Overview

The application consists of:
- **Frontend**: Plain HTML, JavaScript, and Tailwind CSS
- **Backend**: Node.js server
- **Database**: PostgreSQL, Stores bus stop and schedule data imported from GTFS

All components are deployed to Azure cloud services.
